# Personal portfolio
## Descrição do Projeto
Este repositório contém o código HTML e CSS da minha página pessoal, criada para apresentar meu perfil como desenvolvedor. A página inclui:

### Cabeçalho: 
Contém o meu nome de forma abreviada, "J.Piornedo", e um menu de navegação com links para meus perfis no LinkedIn, GitHub e Instagram, representados por ícones.

### Corpo Principal: 
Inclui uma seção de texto com uma breve apresentação sobre mim, destacando minhas habilidades em Python, HTML, CSS e Java, e uma foto de perfil.

### Rodapé: 
Apresenta ícones das tecnologias principais que utilizo, como Python, HTML, CSS e JavaScript.

## Estilização e Layout
Reset de CSS: Remove margens e preenchimentos padrão.
### Fonte Personalizada: 
Utiliza a fonte "Callite".
### Layout: 
Usa CSS Grid e Flexbox para organizar os elementos da página.
### Responsividade: 
Estilos ajustados para diferentes tamanhos de tela, garantindo uma boa visualização em dispositivos móveis e desktops.
### Objetivo
O objetivo desta página é servir como um cartão de visita online, permitindo que visitantes aprendam mais sobre mim, minhas habilidades e onde podem me encontrar online.
